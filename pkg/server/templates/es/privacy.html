Esta Política de Privacidad describe cómo este gateway autohospedado Liferay Tunnel (lfr-tunneld) procesa sus datos.

1. Información que Recopilamos y Procesamos

  • Direcciones IP: Procesadas estrictamente para enrutar paquetes y aplicar protecciones de seguridad contra abusos.
  • Direcciones de Correo: Procesadas para la verificación de cuentas, notificaciones de aprobación e inicios de sesión por enlace mágico.
  • Registros de Auditoría: Registra acciones administrativas, creaciones de tokens e infracciones de seguridad localmente en la base de datos.

2. Seguridad y Almacenamiento de Datos

Los Tokens de Acceso Personal (PAT) se almacenan en el servidor mediante hashes criptográficos seguros SHA-256. Todos los datos se almacenan en una base de datos SQLite local y privada, y nunca se comparten, venden ni transmiten a servidores externos.
